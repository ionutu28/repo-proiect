1️⃣ Game Tetris C++

Description:
I developed a Tetris game in C++ as part of my practice in object-oriented programming and game logic implementation. The project includes core gameplay mechanics such as piece generation, movement, rotation, collision detection, line clearing, and score tracking. I also focused on improving the graphical flow and responsiveness of the game.

Context:
This project was created to strengthen my understanding of OOP concepts, real-time game logic, and algorithmic thinking through a classic game implementation.

Technologies:
C++, Visual Studio Code

Problem it solves:
The project simulates the classic Tetris gameplay experience while demonstrating how object-oriented design and collision detection can be used to manage dynamic game elements efficiently.
